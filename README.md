# Vue-wheel - 一个简易的UI组件
## 安装
使用本框架前，请在css中开启border-box
```coffeescriptliterate
*{box-sizing: border-box;}
```
## 问题
### 遇到的第一个问题，workspace.xml文件报错
```
Element option is not allowed here
```
解决方法：重启idea
File -> Invlidte Cches
### 遇到的第二个问题
```
top-level selectors may not contain the parent selector "&".
```
解决方法：&必须嵌套在scss规则内
